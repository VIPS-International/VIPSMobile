# VIPS Mobile WASM SQL

Is your mobile workforce bogged down in manual processes and paperwork 
resulting in poor field execution and delays? Do they have the information 
they need to make smart, quick decisions?


Can your managers track field activities and communicate with teams in an 
instant?

VIPs Mobile is an intuitive application that allows your remote teams to 
easily complete and submit their daily work and activities while in the field. 

VIPs Mobile also allows communication with your team through voice, video 
and written messages. 

VIPs Mobile includes a self managed document library which ensures that 
your field team always have up to date resources at their fingertips. 
Content can include PDF, Excel, Power Point, Videos, YouTube, Images and 
much more.


## Scope of this Repository

This project comprises:

- The client-side JavaScript, HTML, and CSS for the VIPS Mobile front-end
- Build and configuration files needed to produce the admin console bundle

It **does not** contain:

- VIPS Mobile backend/server code
- Any other internal VIPS International PTY. LTD. tools or services

Those components are developed and licensed separately under 
VIPS International PTY. LTD.’s proprietary terms.
